## Описание

Небольшое консольное приложение для управления задачами прямо в терминале. 

Позволяет создавать, редактировать, просматривать и удалять задачи, а также отслеживать их статусы и получать статистику.

## Возможности

- Создание задач с заголовком и описанием
- Просмотр списка всех задач или фильтрация по статусу
- Детальный просмотр конкретной задачи
- Редактирование заголовка и описания задачи
- Управление статусами: pending → in_progress → completed
- Поиск задач по ключевым словам
- Статистика по задачам
- Удаление задач
- Хранение данных в JSON файле

## Установка

```bash
# Клонировать репозиторий и установить зависимости
go mod download

# Собрать приложение
go build -o todo

# Или запустить напрямую
go run main.go
```

## Использование

#### Создание задачи

```bash
# Создать задачу только с заголовком
todo add "Купить продукты"

# Создать задачу с заголовком и описанием
todo add "Написать отчёт" "Подготовить отчёт для руководства"
```

#### Просмотр задач

```bash
# Показать все задачи
todo list

# Показать только ожидающие задачи
todo list --status pending
todo list -s pending

# Показать задачи в работе
todo list --status in_progress

# Показать выполненные задачи
todo list --status completed
```

#### Детальный просмотр задачи

```bash
# Показать полную информацию о задаче с ID 5
todo show 5
```

#### Редактирование задачи

```bash
# Изменить заголовок
todo edit 14 --title "Новый заголовок"
todo edit 14 -t "Новый заголовок"

# Изменить описание
todo edit 5 --description "Новое описание"
todo edit 5 -d "Новое описание"

# Изменить оба поля
todo edit 7 -t "Новый заголовок" -d "Новое описание"
```

#### Управление статусами

```bash
# Начать выполнение задачи (pending → in_progress)
todo start 3

# Завершить задачу (in_progress → completed)
todo complete 7
```

#### Поиск задач

```bash
# Поиск по ключевому слову (регистронезависимый)
todo search "отчёт"
todo search "продукты"
```

#### Статистика

```bash
# Показать статистику по всем задачам
todo stats
```

#### Удаление задачи

```bash
# Удалить задачу по ID
todo delete 8
```

### Справка

```bash
# Общая справка
todo --help

# Справка по конкретной команде
todo list --help
todo add --help
```
